== Description ==
Turn on your login page. Leave it with a more modern face using the Beautiful Login Page. Simple and easy to use.

== Installation ==
From your WordPress dashboard

1. #Visit Plugins > Add New
2. #Search for \"Beautiful Login page\"
3. #Activate Beautiful Login page from your Plugins page
4. #Click on the new menu item \"BLP\" and configure your Beautiful Login Page!
